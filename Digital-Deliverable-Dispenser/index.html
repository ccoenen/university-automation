    Howto

    This is designed to ensure the deliverables will not be changed without anyone noticing.

    Steps to perform:

    • (optional but highly recommended) generate your folder structure on this site by filliing in your details and clicking "download empty zip".
    • Add your own files to
    • Create a single zip file. When in doubt (or when problems occur), try 7-zip instead. In any case, regular zip is your target format.
    • Create a SHA-256 checksum for the resulting zip file. This can be done in a number of different ways:
      • On Mac OSX and Linux, you can run sha256sum <filename>. The result will be your checksum.
      • On Windows, the same can be done with certUtil -hashfile "<filename>" SHA256.
      • Alternatively, Nextcloud's file context menu has a "checksum" tab where you can select "SHA256" from the dropdown menu
    • E-Mail this Hash to your coaches.
    • Upload your file to Nextcloud
